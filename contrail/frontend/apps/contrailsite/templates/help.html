What is Contrail?

Contrail tracks cloud prices by pulling virtual machine pricing data from AWS and MS Azure into a data warehouse to be able to answer common questions about VM pricing. Users can compare VM prices across multiple regions and cloud providers, compare payment options, and track the cost of a particular VM over time. Users can also track the change of pricing when changing a VM’s characteristics, such as increasing network bandwidth, doubling RAM, etc.





Now, what is a virtual machine?

A virtual machine (VM) is a computer file that behaves like an actual computer. It gives you the flexibility of virtualization without having to buy and maintain the physical hardware that runs it, while being sandboxed from the rest of your system. This creates an ideal environment for testing other operating systems such as beta releases, accessing virus-infected data, creating backups, running software on operating systems it wasn’t designed for, and running applications that require a greater computing capacity. Each VM has its own virtual hardware, and multiple VMs can run concurrently on the same physical computer.

Cloud Providers

As of right now, Contrail pulls pricing data of VMs from both AWS and MS Azure.




AWS

Amazon Web Services is a subsidiary of Amazon that provides on-demand cloud computing platforms on a pay-as-you-go basis. It’s the world’s most comprehensive and broadly adopted cloud platform, offering over 165 fully featured services (over 40 of which that aren’t available anywhere else) from data centers globally. AWS has the largest, most dynamic ecosystem and is architected to be the most flexible and secure cloud computing environment available today.



Amazon EC2

Amazon EC2 (Elastic Compute Cloud) is a web service providing secure, resizable, compute capacity in the cloud. It’s benefits include elastic web-scale computing, complete control of the EC2 instances, flexible cloud hosting services, integration with most other AWS services, reliability, and robust security. Inexpensive purchasing options are also available and it’s easy to get started!


Learn more about AWS, Amazon EC2, and Amazon EC2 pricing.



MS Azure

Microsoft Azure is an ever-expanding set of cloud computing service created by Microsoft for building, testing, deploying, and managing applications and services through Microsoft-managed data centers. It prides itself on being the only consistent hybrid cloud on the market, and is extremely productive, intelligent, and trusted. MS Azure has the largest portfolio in the industry, with more global regions than any cloud provider and markets their services as the cloud for all that finds answers to any business challenge with a scalable, cost-effective solution.



Azure VM

Azure VM is an on-demand, scalable computing resource that gives you the flexibility of virtualization while keeping your budget in check with low-cost, per-second billing where you only pay for the compute time you use. It has the capacity to scale from one to thousands of VM instances in minutes, and provides you with recommendations for high availability, security, performance for all your VMs with other services provided in Azure.


Learn more about MS Azure, Azure VM, and Azure VM pricing.




Payment Plans



On Demand Pricing

On-Demand pricing lets you pay for compute capacity by the hour or second with no long-term commitments or upfront payments. This allows you to increase or decrease your compute capacity depending on the demands of your application and relieves you from the costs and complexities of planning, purchasing, and maintaining hardware. As a result, this transforms the generally large fixed costs into smaller variable costs as you only pay for what you use.


*MS Azure uses the term “Pay-as-you-go” for On-Demand pricing but for standardization purposes, we generalize to On-Demand.



Recommended for:

  • Users who prefer low cost and flexibility of VMs without any up-front payment or long-term commitment.

  • Applications with short-term, spiky, or unpredictable workloads that cannot be interrupted.

  • Applications being developed or tested on VMs for the first time.


Learn more about AWS On-Demand and MS Azure Pay-as-you-go pricing.



Reserved Pricing

Reserved pricing provides you with significant discounts (amount varies by provider) in comparison to On-Demand pricing by pre-paying for either 1-year or 3-year terms of VMs, which allows you to get a discount on the resources you use in both AWS and MS Azure.


Specifically in AWS you have the choice between a Standard or Convertible Reserved pricing. With Standard Reserved pricing, you can receive up to a 75% discount and have the flexibility to alter the Availability Zones, instance sizes, and networking types. Convertible Reserved pricing allows you to have further flexibility, such as having the capacity to use different instance families or operating systems, and provides up to a 54% discount. Additionally with either of those reservations, you can choose to pay with either All Upfront, Partial Upfront, or No Upfront payments. The All Upfront option requires you to pay for the entire Reserved term with one payment, and provides you with the largest discount compared to On-Demand pricing. With Partial Upfront, you are required to pay a low upfront payment and then charged a discounted hourly rate for the VM of your choice for the duration of the Reserved term. With the No Upfront option, no upfront payment is required and a discounted hourly rate is charged for the entire duration of the Reserved term.



Recommended for:

  • Applications with steady state usage.

  • Applications that may require reserved capacity.

  • Customers who want budget predictability and minimize risk.

  • Customers that can commit to using a VM over a 1-year or 3-year term to reduce their total computing costs.


Learn more about AWS and MS Azure reserved pricing.



Spot Pricing

Spot pricing allows you to bid for unused capacity in a cloud provider’s data center. The prices are set by the cloud provider and adjusted gradually based on long-term trends in supply and demand for Spot VM capacity. You pay the Spot price that’s in effect for the time period your Vs are running, and can save up a significant amount when compared to On-Demand pricing, but if someone else bids higher than you, your VM will be taken away. With AWS Spot VMs can also run for a predefined duration (hourly increments up to 6 hours) with a discount of up to 30 - 50% compared to On-Demand pricing. VMs in MS Azure are billed per-second rounded down to the last minute.


*MS Azure uses the term “low-priority” VMs for Spot pricing but for standardization purposes, we generalize to Spot.



Recommended for:

  • Applications that have flexible start and end times.

  • Applications that are only feasible at very low compute prices.

  • Users with urgent computing needs for large amounts of additional capacity.

  • Large-scale solutions that are in development.

  • Large-scale load testing and regression testing.

  • Supplement regular dedicated VMs (when available, jobs can scale and complete quicker for lower cost and when unavailable, the baseline of dedicated VMs remain).


Learn more about AWS Spot and MS Azure Low Priority pricing.




Features



Find Prices

To find and sort through prices of VM’s, navigate to the Get Prices page. To help you get started on finding the right VM for you, click here for a quick guide! The sidebar enables you to toggle cloud providers, regions, operating system, payment plans, memory, vCPUs, etc to your liking. You can customize the number of entries shown per page of the table. You can also search for a particular instance or any keyword using this search bar. Clicking on any of the column headings will sort the results in according to that category.



Compare Prices

In order to compare prices of certain VMs in a more readable manner, the compare button allows you to choose up to three VMs of your choice and view them in a separate table.



Detailed Instance Page

To view more detailed information about a particular VM, click on the instance type to be led to a detailed page that shows information about the different payment plans available and specifications about the instance such as SKU, clock speed, and instance family. The bottom of the page includes a graph showing the pricing trends for the instance with the different payment options. You can toggle the options to view only one to all payment options.



Cloud Storage

To view cloud storage pricing, navigate to the “Cloud Storage” tab. This table has the same sorting and searching properties the instance pricing table has.
